constructor

  • new ParticleScaleNode(mode: number, usesCycle: boolean, usesPhase: boolean, minScale?: number, maxScale?: number, cycleDuration?: number, cyclePhase?: number): ParticleScaleNode

  • Overwrites ParticleNodeBase.constructor

    • Defined in animators/nodes/ParticleScaleNode.ts:41

    Creates a new ParticleScaleNode

    Parameters

    • mode: number

      Defines whether the mode of operation acts on local properties of a particle or global properties of the node.

    • usesCycle: boolean

      Defines whether the node uses the cycleDuration property in the shader to calculate the period of animation independent of particle duration. Defaults to false.

    • usesPhase: boolean

      Defines whether the node uses the cyclePhase property in the shader to calculate a starting offset to the animation cycle. Defaults to false.

    • Default value minScale: number = 1

      Defines the default min scale transform of the node, when in global mode. Defaults to 1.

    • Default value maxScale: number = 1

      Defines the default max color transform of the node, when in global mode. Defaults to 1.

    • Default value cycleDuration: number = 1

      Defines the default duration of the animation in seconds, used as a period independent of particle duration when in global mode. Defaults to 1.

    • Default value cyclePhase: number = 0

      Defines the default phase of the cycle in degrees, used as the starting offset of the cycle when in global mode. Defaults to 0.

    Returns ParticleScaleNode
